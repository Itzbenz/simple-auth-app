# How to run
- ~~use your leg~~
- `apt -y install php php-{cli,gd,mysql,pdo,mbstring,tokenizer,bcmath,xml,fpm,curl,zip}` or download these extensions
- `composer install`
- `./vendor/bin/sail up`







```bash
git clone https://github.com/Itzbenz/simple-auth-app.git
cd simple-auth-app
composer install
./vendor/bin/sail up
```
